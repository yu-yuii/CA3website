Recognising online scams

Different types of online scams

Malware

Malware is designed to disrupt, steal, or gain unauthorized access to a computer system.

Phishing

Deceptive emails or websites designed to trick users into revealing sensitive information or installing malware.

Social Engineering

Psychologically manipulate users into exposing confidential information or granting unauthorized access to systems.

Unauthorized access refers to gaining entry to a computer system without proper authorization.

Then how can you avoid such scams?

Protect all personal informations

Image by Freepik.com

  • Set a strong secret combination of characters to gain access.
  • Use Biometric authentication to verify if it is you.
  • Add Multi-Factor Authentication to enhances security ,like a password and a one-time code sent to your phone.

Update software regularly

Image by static.vecteezy.com

  1. Software updates improves security patches that fix vulnerabilities dealt by attackers.
  2. Updates can help improve your devices system stability and performance which reduces the risk of crashes and errors.

Secure browsing

Image by freepik.com

  • Use caution when clicking links, downloading files, or entering personal information online.
  • Verify Website's URL before enetering your sensitive informations.
  • Do not click on pop-ups that are too good to be true (eg. free prizes and warnings for bank transfer).
  • Use secure browsers such as Inprivate window mode on Microsoft or FireFox for better security.
  • Enable HTTPS-Only mode- avoid using sites with only HTTP.
